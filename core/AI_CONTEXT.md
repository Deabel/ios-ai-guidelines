# AI Context

These rules define how AI coding agents should generate Swift code for
iOS projects.

Primary technologies:

Swift SwiftUI Swift Structured Concurrency Combine (only when reactive
streams are required)

Architecture pattern:

MVVM + Service Layer

Before generating code, the AI must follow the rules defined in:

architecture.md swiftui_rules.md concurrency_rules.md combine_rules.md
naming_rules.md anti_patterns.md
